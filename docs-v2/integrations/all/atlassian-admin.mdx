## Common Scopes

| Scope                        | Description       |
| ----------------------------------- | -------------------------------------------------------------------------- |
| `read:atlassian-admin-user`         | Read user information                                                      |
| `read:atlassian-admin-work`         | Read Atlassian Admin work items (issues, projects, etc.)                   |
| `write:atlassian-admin-work`        | Create and update Atlassian Admin work items                               |
| `read:classification-levels:admin`  | Get all classification levels and specific level by org ID                 |
| `write:classification-levels:admin` | Create, edit, publish, archive, restore, and reorder classification levels |
| `read:tokens:admin`                 | Retrieve API keys and tokens in the organization                           |
