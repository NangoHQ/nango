## Setup guide

<Steps>
  <Step title="Create a Recruiterflow account">
    Go to [Recruiterflow's signup page](https://recruiterflow.com/sign-up) and create an account. You can use the free trial for testing.
  </Step>
  <Step title="Create & configure an API key">
    <Tip>Generating an API key [requires contacting Recruiterflow support](https://help.recruiterflow.com/en/articles/3671870-use-recruiterflow-api-to-build-custom-careers-page).</Tip>
  </Step>
  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart).
  </Step>
</Steps>

<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>
