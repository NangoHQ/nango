## Setup guide

<Steps>
  <Step title="Create a Recruiterflow account">
    Go to [Recruiterflow's signup page](https://recruiterflow.com/sign-up) and create an account. You can use the free trial for testing.
  </Step>
  <Step title="Create & configure an API key">
    1. Log in to your Recruiterflow account
    2. Navigate to Settings > API Keys
    3. Click "Create New API Key"
    4. Give your API key a name and copy the generated key
    5. You will need this API key when configuring your integration in Nango
  </Step>
  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart).
  </Step>
</Steps>

## API gotchas

- Recruiterflow uses API key authentication rather than OAuth
- API keys have a 30-day expiration by default
- Rate limits are applied per API key
- All API requests must include the API key in the Authorization header
