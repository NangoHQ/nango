# Overview

To authenticate with Jobvite, you need:
1. **API Key** - A key that grants Nango permission to interact with Jobvite's APIs resources and services.
2. **Secret Key** - A key that validates the API key and grants access to Jobvite's services.

This guide will walk you through obtaining these credentials within Jobvite.

### Prerequisites:

- You must have a valid Jobvite account login

### Instructions:

#### Step 1: Getting Your API Key and Secret Key
- In order to access the Jobvite webservices you will need to be issued an API key and secret
both provided by Jobviteâ€™s Customer Success team.

- To request API credentials, file a support ticket at: [Jobvite Support](http://www.jobvite.com/support/submit-a-case/)

#### Step 3: Enter credentials in the Connect UI

Once you have your **API Key** and **Secret Key**:
1. Open the form where you need to authenticate with Jobvite.
2. Enter your **API Key** and **Secret Key** in their respective fields.
3. Submit the form, and you should be successfully authenticated.
