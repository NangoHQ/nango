# Overview

To authenticate with Pandadoc, you will need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Pandadoc resources and services.

This guide will walk you through generating your **API Key** within Pandadoc.

### Prerequisites:

- You must have an account with Pandadoc.
- To use a **Production key**, you need to be on the Enterprise plan and get a Production API activated by [Pandadoc's team](https://developers.pandadoc.com/reference/production-api-key).

### Instructions:

#### Step 1: Generating your API Key
1. Sign in to your [Pandadoc](https://app.pandadoc.com/) account.
2. Navigate to the [Configuration](https://app.pandadoc.com/a/#/api-dashboard/configuration) page.
<img src="/integrations/all/pandadoc-api-key/configuration.png" />
3. On this page, you can generate either of the two API key types: **Sandbox Key or Production Key**.


#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Pandadoc (API Key).
2. Enter your **API Key** in its respective field.
3. Submit the form, and you should be successfully authenticated.


<img src="/integrations/all/pandadoc-api-key/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Pandadoc (API Key).
