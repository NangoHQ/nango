  <Tab title="ðŸš¨ API gotchas">
    - Workday provides two APIs, a REST and a SOAP one. Nango supports them both.
    - Workday requires a user-specific `hostname` and `tenant` for the API base URL.
    - Access to most Workday's API documentation is typically restricted to Workday customers and partners.
