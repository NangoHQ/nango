# Overview

To authenticate with Instantly, you need an API key. This guide will walk you through generating an API key and connecting your Instantly account.

### Prerequisites:

- You must have an Instantly account
- Note: The V2 API will be required starting in 2025 when V1 is deprecated

### Step 1: Generate an API Key

1. Log into your [Instantly account](https://app.instantly.ai/app/settings/integrations)
2. Go to **Integrations**
3. Click on **API Keys** in the left sidebar
4. Click the **Create API Key** button
5. Enter a name for your API key (this is for display purposes only)
6. Select the API scopes you want this key to have access to
7. Click **Create**

<img src="/integrations/all/instantly/form.png" />

8. Copy your API key and store it securely - it will only be shown once and cannot be recovered

<Warning>
For security reasons, the API key will only be displayed once during creation. Make sure to save it in a secure location as there is no way to recover it if lost.
</Warning>

### Step 2: Connect Your Instantly Account

To authenticate using your API key, navigate to the Instantly authentication form in Nango and enter your API key.

<img src="/integrations/all/instantly/form.png" />

Click "Connect" to complete the authentication.
