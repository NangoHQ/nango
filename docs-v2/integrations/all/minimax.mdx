## Setup guide

1. **Obtain your Group ID**:
   - Log in to your MiniMax account.
   - Navigate to the **Account** tab and select **Your Profile** from the left menu.
   - Locate the **GroupID** field in the basic information. This is a 19-digit number. Copy it for use in your API requests.

2. **Generate an API Key**:
   - In the left menu, click on **API Keys**, then select **Create New Secret Key**.
   - Enter a name for the key in the provided field, then click **Create**.
   - A dialog box will appear. Use the copy icon to securely save the API key, as it will not be displayed again.

## API gotchas

-   **Group ID is required**: Every API request must include the Group ID in the headers.
-   **API key security**: Ensure your API key is stored securely, as it cannot be retrieved after creation.
-   **Rate limits**: MiniMax enforces rate limits on API requests. Check the [API documentation](https://api.minimaxi.com/docs/rate-limits) for details.
