## API gotchas

- Quickbase has different types of Tokens for its API.
- If you use a user token, you don't need to use an app token
- Quickbase uses `API_KEY` auth mode with `Authorization: QB-USER-TOKEN {TOKEN}` as a request header to access different endpoints using the user token.
