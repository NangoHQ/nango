# Overview

To authenticate with Recruiterflow, you need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Recruiterflow resources and services.

This guide will walk you through generating the API Key within Recruiterflow.

### Prerequisites:

- You must have a Recruiterflow account.

### Instructions:

#### Step 1: Generating Your API Key

Generating an API key [requires contacting Recruiterflow support](https://help.recruiterflow.com/en/articles/3671870-use-recruiterflow-api-to-build-custom-careers-page).

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Recruiterflow.
2. Enter the **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/integrations/all/recruiterflow/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Recruiterflow.
