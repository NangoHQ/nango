## Setup guide

<Steps>
  <Step title="Create a Trafft account">
    Go to [Trafft signup page](https://www.trafft.com/signup/) and create an account if you don't have one already.
  </Step>
  <Step title="Get your OAuth credentials">
    1. Log in to your Trafft admin panel at `https://your-subdomain.admin.trafft.com`
    2. Navigate to **Features** > **Integrations** > **API Settings**
    3. Here you will find your:
       - API URL
       - Client ID
       - Client Secret
    4. You will need these credentials and your subdomain from the API URL when configuring your integration in Nango.
  </Step>
  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart).
  </Step>
</Steps>
