## Setup guide

<Steps>
  <Step title="Login/Sign-up to your xAI Account">
    Go to [xAI Account](https://accounts.x.ai/sign-in) and login or sign up using your credentials.
  </Step>

  <Step title="Generate xAI API key">
    Click _Go to API console_ button in the lower left. 

    In the left sidebar, click _API keys_.

    Click _Create API key_.

    Enter a name for the key, select endpoints and the models.

    Click _Save_.
    
    Copy the API key—you'll need to provide it in Nango when setting up your xAI connection.
  </Step>

  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart).
  </Step>
</Steps>

## API gotchas

- Keeping your xAI API key secure is essential. Unauthorized access allows others to use your project's quota, potentially resulting in quota loss or unexpected charges for billed projects. It also exposes your tuned models and files to unauthorized use.
