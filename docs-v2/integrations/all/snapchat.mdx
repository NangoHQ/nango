                ```bash
                  -H "Connection-Id: <CONNECTION-ID>"
                ```
            </Tab>

            <Tab title="Node">

            Install Nango's backend SDK with `npm i @nangohq/node`. Then run:

            ```typescript

            const nango = new Nango({ secretKey: '<NANGO-SECRET-KEY>' });

            const res = await nango.get({
                endpoint: '/v1/me',
                providerConfigKey: '<INTEGRATION-ID>',
                connectionId: '<CONNECTION-ID>'
            });

            console.log(JSON.stringify(res.data, null, 2));
            ```
            </Tab>

      </Step>
      <Step title="Obtain your client credentials">
        1. After creating the app, copy the **Snap Client ID** and **Snap Client Secret Key** that Snapchat will generate. Youâ€™ll need these when configuring your integration in Nango.

    ## Common Scopes
    | Scope | Description |
    | ----- | ----------- |
    | `snapchat-marketing-api` | Read and write to the Snapchat Marketing APIs |
    | `snapchat-offline-conversions-api` | Read and write to the Snapchat Conversions APIs |
