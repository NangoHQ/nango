    <Steps>
      <Step title="Create an integration">
        In Nango ([free signup](https://app.nango.dev)), go to [Integrations](https://app.nango.dev/dev/integrations) -> _Configure New Integration_ -> _Zoho_.
      </Step>
      <Step title="Authorize Zoho Bigin">
        Go to [Connections](https://app.nango.dev/dev/connections) -> _Add Test Connection_ -> _Authorize_, then log in to Zoho Bigin. Later, you'll let your users do the same directly from your app.
      </Step>
      <Step title="Call the Zoho Bigin API">
        Let's make your first request to the Zoho Bigin API (fetch a list of users in the organization). Replace the placeholders below with your [secret key](https://app.nango.dev/dev/environment-settings), [integration ID](https://app.nango.dev/dev/integrations), and [connection ID](https://app.nango.dev/dev/connections):
        <Tabs>
            <Tab title="cURL">
                ```bash
                curl "https://api.nango.dev/proxy/bigin/v2/users?per_page=10" \
                  -H "Authorization: Bearer <NANGO-SECRET-KEY>" \
                  -H "Provider-Config-Key: <INTEGRATION-ID>" \
                  -H "Connection-Id: <CONNECTION-ID>"
                ```
            </Tab>

            <Tab title="Node">

            Install Nango's backend SDK with `npm i @nangohq/node`. Then run:

            ```typescript
            import { Nango } from '@nangohq/node';

            const nango = new Nango({ secretKey: '<NANGO-SECRET-KEY>' });

            const res = await nango.get({
                endpoint: '/bigin/v2/users',
                providerConfigKey: '<INTEGRATION-ID>',
                params: {
                    per_page: 10
                },
                connectionId: '<CONNECTION-ID>'
            });

            console.log(JSON.stringify(res.data.users, null, 2));
            ```
            </Tab>

        </Tabs>

        Or fetch credentials dynamically via the [Node SDK](/reference/sdks/node#get-a-connection-with-credentials) or [API](/reference/api/connection/get).

      </Step>
    </Steps>

    ## Common Scopes
    | Scope                              | Description                                                                  |
    | ---------------------------------- | ---------------------------------------------------------------------------- |
    | `ZohoBigin.users.ALL`            | Full access to manage users in the organization.                             |
    | `ZohoBigin.org.ALL`              | Full access to organization details.                                         |
    | `ZohoBigin.settings.ALL`         | Manage all settings (modules, roles, profiles, fields, layouts, tags, etc.). |
    | `ZohoBigin.modules.contacts.ALL` | Full access to contacts module.                                              |
    | `ZohoBigin.modules.accounts.ALL` | Full access to accounts (companies) module.                                  |
    | `ZohoBigin.modules.products.ALL` | Full access to products module.                                              |
    | `ZohoBigin.bulk.ALL`             | Manage bulk read/write operations.                                           |
    | `ZohoBigin.notifications.ALL`    | Full access to manage notifications (subscriptions).                         |
