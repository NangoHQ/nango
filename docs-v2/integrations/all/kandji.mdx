## API gotchas

- Kandji uses API_KEY auth mode with `authorization: Bearer API_KEY` in the request header to access different endpoints.
- Kandji enforces a general [API rate limit](https://support.kandji.io/kb/kandji-api#api-rate-limit) of 10,000 requests per hour per tenant.
