# Overview

To authenticate with Terraform, you need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Terraform resources and services.

This guide will walk you through generating the API Key within Terraform.

### Prerequisites:

- You must have a Terraform account.

### Instructions:

#### Step 1: Generating Your API Key

Generating an API key [following Terraform's guide](https://developer.hashicorp.com/terraform/cloud-docs/users-teams-organizations/api-tokens).

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Terraform.
2. Enter the **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/integrations/all/terraform/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Terraform.
