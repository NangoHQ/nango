# Overview

To authenticate with JobDiva, you will need:
1. **Client ID** - Provided by JobDiva, it is used in conjunction with your API user credentials to generate an access token for authenticating API requests.
2. **Username** - The username associated with your JobDiva API user.
3. **Password** - The password for your JobDiva API user.

This guide will walk you through obtaining your **Client ID** and generating your **Username** and **Password** within JobDiva.

### Prerequisites:

- You must have a JobDiva account.

### Instructions:

#### Step 1: Obtaining your Client ID

Contact [JobDiva support](mailto:support@jobdiva.com) to find your client ID.

#### Step 2: Creating an API User
1. Log in to your [JobDiva account](https://login.jobdiva.com/).
2. Navigate to **Settings** > **My Team** > **Add User**.
3. Add a new user.
4. Select the **Only allow access to JobDiva API Calls** permission for the user.

#### Step 3: Enter credentials in the Connect UI

Once you have your **Client ID**, **Username** and **Password**:
1. Open the form where you need to authenticate with JobDiva.
2. Enter your credentials in their designated fields.
3. Submit the form, and you should be successfully authenticated.

You are now connected to JobDiva.
