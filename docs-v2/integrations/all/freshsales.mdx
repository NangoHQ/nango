## API gotchas

- Freshsales uses API_KEY auth mode with `Authorization: Token token= API_KEY` in the request header to access different endpoints.
- In Freshsales, the rate limit is on an account level. It is irrespective of number of users. For details check on [freshsales rate limit](https://support.freshsales.io/en/support/solutions/articles/223406-does-freshsales-have-api-request-limits-for-an-account-)
