## API gotchas

-   Fiserv, supports different authentication methods for different products.
-   When creating a new Fiserv connection in Nango, please provide your `API key` as your client Id and `API secret` as your client secret.
-   To obtain the Fiserv `hostUrl`, navigate to the API key section of your workspace. If your `hostUrl` from Fiserv is `https://card-sandbox.api.fiservapps.com`, add your `hostUrl` in Nango as `card-sandbox.api.fiservapps.com`. For more details check [request URL](https://developer.fiserv.com/product/BankingHub/docs/?path=docs/get-started.md&branch=main#request-url)
