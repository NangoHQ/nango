## API gotchas

-   Access tokens expire after 2 hours and must be refreshed using the refresh token
-   Refresh tokens are valid indefinitely but are invalidated when exchanged for a new access token
-   The OAuth 2.0 flow uses the Authorization Code Grant type w/o PKCE
-   No scope is needed when setting up the integration
