# Overview

To authenticate with Recruit CRM, you need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Recruit CRM resources and services.

This guide will walk you through generating the API Key within Recruit CRM.

### Prerequisites:

- You must have a Recruit CRM account.

### Instructions:

#### Step 1: Generating Your API Key

1. Log in to your Recruit CRM account
2. Navigate to Settings > API Keys
3. Click on "Generate New API Key"
4. Copy the generated API key immediately as it won't be shown again

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Recruit CRM.
2. Enter the **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Recruit CRM.
