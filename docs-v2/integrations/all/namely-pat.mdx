## API gotchas

-   PATs have an expiration time of 2 years.
-   Namely API limits GET `/profiles` to 100 requests per minute, returning HTTP 406 if exceeded.
-   Namely (PAT) uses API_KEY auth mode with `authorization: Bearer API_KEY` in the request header to access different endpoints.
