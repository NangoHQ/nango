## Setup guide

To get started with the Incident.io integration:

1. Sign up for a Teams tier plan or higher at [Incident.io](https://incident.io)
2. Generate an API key in your [Incident.io settings](https://app.incident.io/settings/api-keys)
3. Use the API key to authenticate with Nango

For detailed connection instructions, see our [connection guide](/integrations/all/incident-io/connect).

<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>

## API gotchas

- API keys must be passed in the Authorization header as `Bearer <api_key>`
- API keys can have different scopes - make sure to grant the required permissions when creating the key
