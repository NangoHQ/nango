## Setup guide

1. Go to your **Admin panel** in Grammarly Enterprise or Education.
2. Open the **Organization** tab and choose **SCIM** in the **Configurations** section.
3. Click **Generate API Key** in the upper-right corner.
4. Copy your **API Key** - you'll need this to authenticate with the SCIM API.

<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>

## API gotchas

-   Only Grammarly Enterprise and Education admins have access to generate SCIM API keys.
-   The API uses standard SCIM 2.0 endpoints for user and group management.
