## API gotchas

- SAP Concur uses **OAuth 2.0 client credentials** for authentication.
- The base URL is determined by the **geolocation** of your account (`us`, `eu`, `cn`). Ensure you use the correct base URL for authentication and API requests.
- The `companyUUID` and `companyRequestToken` are required for some operations, ensure they are correctly obtained.
