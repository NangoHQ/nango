## API gotchas

- The `audience` parameter is required in token requests and varies depending on the endpoint, for example, use `https://api.pax8.com` for public APIs and `api://provisioning` when requesting a token for provisioning endpoints.
- Access tokens have a 1-day lifetime (86,400 seconds)
- Only Partner Admin and Primary Partner Admin roles can generate API credentials
