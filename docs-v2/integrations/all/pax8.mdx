## API gotchas

- Uses Client Credentials OAuth2 flow, not traditional authorization code flow
- Requires `audience` parameter in token requests, varies by endpoint (e.g., `https://api.pax8.com` for public APIs)
- Access tokens have a 1-day lifetime (86,400 seconds)
- Only Partner Admin and Primary Partner Admin roles can generate API credentials
- All requests must use HTTPS
