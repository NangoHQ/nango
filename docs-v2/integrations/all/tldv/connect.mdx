# Overview

To authenticate with tl;dv, you need:
1. **API key** - A key that grants Nango permission to interact with tl;dv's APIs resources and services.

This guide will walk you through obtaining your API key within tl;dv.

### Prerequisites:

- You must have an account with tl;dv.
- You must be subscribed to a plan that includes API access.

### Instructions:

#### Step 1: Finding Your API Key
1. Log in to your [tl;dv](https://tldv.io/app/) account.
2. Navigate to the [api Keys](https://tldv.io/app/settings/personal-settings/api-keys) page.
3. Generate a new API Key.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with tl;dv.
2. Enter your **API Key** in its respective field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to tl;dv.
