## Common Scopes

- **ZohoInvoice.fullaccess.all** - Full access to Invoice data
- **ZohoInvoice.invoices.READ** - Read access to invoices
- **ZohoInvoice.invoices.CREATE** - Create invoices
- **ZohoInvoice.invoices.UPDATE** - Update invoices
- **ZohoInvoice.invoices.DELETE** - Delete invoices
- **ZohoInvoice.contacts.READ** - Read access to contacts
- **ZohoInvoice.settings.READ** - Read access to settings

## API gotchas

<ZohoSharedGotchas />

### Invoice-specific gotchas
- Zoho Invoice API has region-specific endpoints. You need to use the correct endpoint based on the user's region.
- The Invoice API has rate limits that vary based on your Zoho Invoice plan.
