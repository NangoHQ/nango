## Setup guide

Clerk is a comprehensive authentication and user management platform that provides a complete suite of authentication tools including sign-up, sign-in, user profiles, and session management.

To integrate Clerk with Nango, you'll need to obtain your Secret Key from the Clerk Dashboard.

## API gotchas

- **API Key Format**: Clerk uses different key prefixes for different environments. Use `sk_test_` keys for development and `sk_live_` keys for production.
- **Rate Limits**: Clerk implements rate limiting based on your subscription plan. Free tier includes 10,000 monthly active users and standard rate limits.
- **API Versioning**: Clerk uses API versioning through headers. The current version is v1, accessible at `https://api.clerk.com/v1`.
- **Authentication**: All API requests require Bearer token authentication using your Secret Key in the Authorization header.
