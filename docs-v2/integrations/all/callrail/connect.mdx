# Overview

To authenticate with Callrail, you will need:
1. **API Key** - A key that grants Nango permission to interact with Callrail's APIs resources and services.

This guide will walk you through finding the API Key within Callrail.

### Prerequisites:

- You must have a registered account with Callrail.

### Instructions:

#### Step 1: Finding Your API Key
1. **Log in** to your Callrail dashboard.
2. Select **Integrations** in the left-hand menu, then select **API Keys**.
3. Click **Create API V3 Key**.

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Callrail.
2. Enter the **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Callrail.
