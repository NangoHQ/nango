# Overview

To authenticate with CheckHQ, you need one key piece of information:
1. **API Token** - Your CheckHQ API Token

This guide will walk you through finding or creating those credentials within CheckHQ.

### Prerequisites:

- You must set up an administrator account with CheckHQ.

#### Step 1: Finding Check API Token

1. The CheckHQ team will provide you an API Key. Securely store it.

#### Step 2: Connect Your Check Account

To authenticate using your API credentials, navigate to the Check authentication form.
Enter your credentials as follows:
- CheckHQ API Token: Your CheckHQ API Token


Click "Connect" to complete the authentication.
