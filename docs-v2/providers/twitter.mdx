## Using Twitter with Nango[](#using-twitter-with-nango "Direct link to Using Twitter with Nango")

Provider template name in Nango: `twitter`  
Follow our [quickstart](/quickstart) to add an OAuth integration with Twitter in
5 minutes.

info

Nango uses Twitter's OAuth 1 flow. As a result, you should not use the "OAuth
2.0 Client ID and Client Secret" but the "Consumer Keys" instead ("API Key" and
"API Key Secret").
