<Steps>
  <Step title="Configure an integration">
    [Sign up](https://app.nango.dev/signup) (free, no credit card) then, on the **Integrations** tab, set up a new integration.
  </Step>
  <Step title="Authorize the API">
    On the **Connections** tab, create a new connection and complete the auth flow. Later, you'll embed the auth flow in your product with code like this:
    ```typescript
    nango.openConnectUI({ onEvent: (event) => { /* handle completion */ } });
    ```
  </Step>
  <Step title="Access the API">
    Retrieve the API credentials from your connection, at scale:

    <Tabs>
        <Tab title="Node">
        ```typescript
        import { Nango } from '@nangohq/node';

        const nango = new Nango({ secretKey: '<NANGO-SECRET-KEY>' }); // Find in your Environment Settings

        const connection = await nango.getConnection(
            '<INTEGRATION-ID>', // Find in your Integrations
            '<CONNECTION-ID>'   // Find in your Connections
        );

        console.log(connection.credentials);
        ```
        </Tab>

        <Tab title="cURL">

        ```bash
        # <NANGO-SECRET-KEY>: Find in your Environment Settings
        # <INTEGRATION-ID>: Find in your Integrations
        # <CONNECTION-ID>: Find in your Connections
        curl -X GET \
        -H 'Authorization: Bearer <NANGO-SECRET-KEY>' \
        'https://api.nango.dev/connection/<CONNECTION-ID>?provider_config_key=<INTEGRATION-ID>'
        ```

        </Tab>
    </Tabs>

    ðŸŽ‰ You're ready to consume any external API!
