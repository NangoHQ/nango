# Overview

To authenticate with Trakstar Hire, you need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Trakstar Hire resources and services.

This guide will walk you through generating the API Key within Trakstar Hire.

### Prerequisites:

- You must have an account with Trakstar Hire.

### Instructions:

#### Step 1: Generating Your API Key

1. Login to your [Trakstar Hire](https://app.hire.trakstar.com/accounts/start-login) account.
2. Navigate to **Settings** > **Advanced Settings** and click **API Key**.
<img src="/integrations/all/trakstar-hire/settings.png" />
3. Click **Generate API Key** to generate a new API Key.
<img src="/integrations/all/trakstar-hire/api_key.png" />
<img src="/integrations/all/trakstar-hire/generated_key.png" />

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Trakstar Hire.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.


<img src="/integrations/all/trakstar-hire/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Trakstar Hire.
