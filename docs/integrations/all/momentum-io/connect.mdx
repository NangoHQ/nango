# Overview

To authenticate with Momentum.io, you will need:
1. **API Key** - A key that grants Nango permission to interact with Momentum.io's API resources and services.

This guide will walk you through generating your **API Key**.

### Prerequisites:

- You must have a Momentum.io account.
- To use the Momentum.io API, your organization needs the API access feature enabled. Please contact the Momentum support team and request that API access be turned on for your workspace.

### Instructions:

#### Step 1: Finding Your API Key
1. Log in to your [Momentum.io account](https://app.momentum.io/).
2. Navigate to the **Integrations** page in your Momentum dashboard.
3. Select **Create API key**.
4. Your **API Key** will be generated.

#### Step 2: Enter credentials in the Connect UI

Once you have your API Key:
1. Open the form where you need to authenticate with Momentum.io.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Momentum.io.
