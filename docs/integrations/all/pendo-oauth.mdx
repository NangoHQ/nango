## API gotchas

- Pendo OAuth uses the `client_credentials` grant type. Tokens are obtained from `https://app.pendo.io/oauth/v1/token` by sending `client_id`, `client_secret`, `grant_type=client_credentials`, and `scope=read:me` as `application/x-www-form-urlencoded` body parameters.
- OAuth `client_credentials` access for Pendo is currently an early-release capability and is not yet covered by Pendo's public documentation. To obtain a client ID and client secret, contact your Pendo account team.
- The main URL used to access the Pendo API corresponds to the region and web address you use when logging into Pendo's UI. The baseUrl is set to `https://app.pendo.io` but you can override it using `baseUrlOverride`.
