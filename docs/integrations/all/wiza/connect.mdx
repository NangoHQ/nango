## Step 1: Getting your API key

1. Log in to your Wiza account and navigate to the [API section](https://wiza.co/app/settings/api).
2. Click the **New API Key** button to generate a new API key.

## Step 2: Enter the key in the Connect UI

1. Open the form where you need to authenticate with Wiza.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.
