# Overview

To authenticate with Lever, you need one key piece of information:
1. **Lever API Key** - Your unique account identifier.

This guide will walk you through finding or creating those credentials within Lever.

### Prerequisites:

- You must set up an account with Lever.

#### Step 1: Finding Lever API Key
1. Log into your Lever account and navigate to the `Integrations and API` page in settings.
2. Click the `API CREDENTIALS` tab. Below the `Lever API credentials`, click `GENERATE NEW KEY`.
3. After adding the scopes for the key and display name, click `GENERATE KEY`.

<img src="/integrations/all/lever-basic/lever-basic.png" />

Important: Copy and store your API Key securely; it will not be visible again once you leave the creation dialog.

#### Step 2: Connect Your Lever Account

<img src="/integrations/all/lever-basic/nango_connect_ui.png" />

To authenticate using your API credentials, navigate to the Lever authentication form.
Enter your credentials as follows:
- Username: Your API Key

Click "Connect" to complete the authentication.
