# Overview

To authenticate with OpenHands, you need:
1. **API Key** - A key that grants Nango permission to interact with OpenHands's APIs resources and services.

This guide will walk you through obtaining your API Key within OpenHands.

### Prerequisites:

- You must have an account with OpenHands.

### Instructions:

#### Step 1: Finding your API Key
1. Log in to your [OpenHands account](https://app.all-hands.dev/).
2. Navigate to [API Settings](https://app.all-hands.dev/settings/api-keys).
3. Select the API Keys tab.
4. Click Create API Key.
5. Give your key a descriptive name (Example: "Development" or "Production") and select Create.
6. Copy the generated API key and store it securely. It will only be shown on
<img src="/integrations/all/open-hands/tokens.png" />


#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with OpenHands.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to OpenHands.
