## API Gotchas

- When testing your Zoom application in the marketplace, Zoom requires you to verify your domain. For OAuth redirects, ensure you use a custom callback URL. This way, Zoom sees your own callback URL. For more details, refer to the [implement API auth guide](/implementation-guides/api-auth/implement-api-auth#5-setup-a-custom-oauth-callback-url-optional).
