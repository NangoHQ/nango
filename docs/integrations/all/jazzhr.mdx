## Access requirements
| Pre-Requisites | Status | Comment |
| - | - | - |
| Paid dev account | ✅ Required | You must have an active JazzHR account with API access enabled. |
| Paid test account | ✅ Required | You must use a real JazzHR account for testing. There is no free test account option. |
| Partnership | ✅ Not required | |
| App review | ✅ Not required | |
| Security audit | ✅ Not required | |

## Setup guide

<Steps>
  <Step title="Retrieve the API key">
    Log in to your JazzHR account.  
    Navigate to _Settings_ > _Integrations_.  
    Copy the API key, you will need it when configuring Nango.
  </Step>
  <Step title="Next">
    Follow the [_Quickstart_](/getting-started/quickstart) to set up your JazzHR integration in Nango.
  </Step>
</Steps>

<Tip>Need help getting started? Get help in the [community](https://nango.dev/slack).</Tip>

## API gotchas

- API requests are limited to 100 requests per minute
- All API endpoints are prefixed with `/api/v1/`
- Date fields should be formatted as `YYYY-MM-DD`
