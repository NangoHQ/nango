## Setup guide

_Set up AgentLine in 2 steps._

<a id="step-1-finding-your-api-key"></a>
### Step 1. Finding your API Key

1. Sign up or log in at [agentline.cloud](https://agentline.cloud)
2. Go to your dashboard and copy your API key (starts with `sk_live_`)

### Step 2. Provision a phone number and agent

After connecting, create an agent and provision a phone number through the AgentLine API:

```bash
# Create an agent
curl -X POST https://api.agentline.cloud/v1/agents \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"name": "My Agent"}'

# Provision a phone number
curl -X POST https://api.agentline.cloud/v1/numbers \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "<AGENT_ID>", "area_code": "313"}'
```

## API gotchas

- AgentLine uses API Key auth — pass as `Authorization: Bearer <key>`
- API keys always start with `sk_live_`
- Calls are handled in **hosted mode** — the server runs the AI voice conversation autonomously
- Poll `GET /v1/calls/<call_id>` for call status, then fetch the transcript via `GET /v1/calls/<call_id>/transcript`
- Available voice presets: `female-1`, `female-2`, `male-1` (or any Cartesia voice UUID)
