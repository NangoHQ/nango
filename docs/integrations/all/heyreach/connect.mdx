# Overview

To authenticate with HeyReach, you need:
1. **API Key** - A key that grants Nango permission to interact with the HeyReach API.

This guide will walk you through generating your API key within HeyReach.

### Prerequisites:

- You must have an account with HeyReach.

### Instructions:

#### Step 1: Generating an API Key
1. Log in to your HeyReach account and navigate to [API Integration](https://app.heyreach.io/app/integrations/public-api/api).
2. Click on **New API key** to create a new **API Key**.
<img src="/integrations/all/heyreach/generate_api_key.png" />


#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with HeyReach.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/integrations/all/heyreach/form.png" style={{maxWidth: "450px" }}/>


You are now connected to HeyReach.
