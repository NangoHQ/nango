# Overview

To authenticate with Hightouch, you will need:
1. **API Key** - A key that grants Nango permission to interact with Hightouch's API resources and services.

This guide will walk you through finding your API Key within Hightouch.

### Prerequisites:

- You must have a registered Hightouch account.
- You must be an Admin user of your Hightouch workspace.

### Instructions:

#### Step 1: Finding Your API Key

1. Log in to your Hightouch workspace and navigate to the **API Keys** tab in the Settings page.
2. Select "Add API key" to create a new API key.
3. Enter a descriptive name for your API key.
4. Copy the generated API key (it will only be displayed once).
5. Store the API key securely.

<Warning>The API key is only displayed once during creation. If you lose it, you'll need to create a new one.</Warning>

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Hightouch.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Hightouch.

### Notes

- API keys provide read/write access to all Hightouch resources, so keep them secure
- If an API key is compromised, immediately revoke it through the Settings page
- API keys follow UUID format: `d74eef25-6fa6-4e6a-a27c-b6c1b02bc470`
