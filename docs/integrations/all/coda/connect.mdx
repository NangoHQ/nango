# Overview

To authenticate with Coda.io, you will need:

1. **API Key** - A key that grants Nango permission to interact with Coda.io's API resources and services.

This guide will walk you through generating the API Key within Coda.io.

## Prerequisites:
- You should have a registered account with Coda.io. Create one on [Coda.io's website](https://coda.io/).
- Permission to generate an **API Key**.

## Step 1: Creating an API Key

1. **Log in** to your Coda.io dashboard.

<img src="" />

2. Click on your **profile image** in the top right corner of the screen.

<img src="/integrations/all/coda/coda_select_profile_icon.png" />

3. Select **Account Settings** from the dropdown menu.

<img src="/integrations/all/coda/coda_select_account_settings.png" />

4. Scroll down to the **API Settings** section and click on the **Generate API Token** button.

<img src="/integrations/all/coda/coda_click_generate_api_token.png" />

5. Enter a name for the API Token. You can optionally **Add restrictions** for the API Token, though for full functionality with Nango, leaving it unrestricted is recommended. Once complete, click **Generate API Token** to save the token.

<img src="/integrations/all/coda/coda_generate_api_token.png" />

6. Copy the API Token and store it securely. Once you leave the page, you wonâ€™t be able to retrieve it again.

<img src="/integrations/all/coda/coda_copy_token.png" />


**Note:** Ensure you store this key securely, as it will not be visible again after leaving the API Key page.

## Step 2: Connect Your Coda.io Account


To authenticate using your API credentials, navigate to the Coda.io authentication form in Nango.

Enter your credentials as follows:
- **API Key**: Paste your API Key from Coda.io.

<img src="/integrations/all/coda/connect_ui.png" />

Click **Connect** to complete the authentication.

**Note:** Ensure the API Key is entered exactly as generated to avoid authentication errors.
