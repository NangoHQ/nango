# Overview

To authenticate with Cal.com (v1), you will need:
1. **API Key** - A key that grants Nango permission to interact with Cal.com's (v1) APIs resources and services.

This guide will walk you through finding or creating the API Key within Cal.com.

### Prerequisites:

- You must have a registered account with Cal.com.

### Instructions:

#### Step 1: Finding Your API Key
1. **Log in** to your Cal.com dashboard.
2. Select **Settings** in the bottom-left corner, then navigate to Developer section and click on **API Keys**.
3. In your API Keys page, click the **Add** button to add a new API Key. A form will appear where you'll need to provide a **Name** and set **key expiration duration**. Once you save these details, an **API Key** will be generated.

<img src="/integrations/all/cal-com-v1/cal_com_api_key.png" />

#### Step 2: Enter credentials in the Connect UI

Once you have the **API Key**:
1. Open the form where you need to authenticate with Cal.com.
2. Enter the **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.


<img src="/integrations/all/cal-com-v1/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Cal.com (v1).
