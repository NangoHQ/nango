  <Tab title="ðŸ“š API Reference">
    The Zendesk Sell API is a REST API that allows you to access and manage your sales data including deals, contacts, organizations, and more.

    ### Common Endpoints

    #### Deals
    - `GET /v2/deals` - Retrieve all deals
    - `GET /v2/deals/:id` - Retrieve a specific deal
    - `POST /v2/deals` - Create a new deal
    - `PUT /v2/deals/:id` - Update an existing deal
    - `DELETE /v2/deals/:id` - Delete a deal

    #### Contacts
    - `GET /v2/contacts` - Retrieve all contacts
    - `GET /v2/contacts/:id` - Retrieve a specific contact
    - `POST /v2/contacts` - Create a new contact
    - `PUT /v2/contacts/:id` - Update an existing contact

    #### Organizations
    - `GET /v2/organizations` - Retrieve all organizations
    - `GET /v2/organizations/:id` - Retrieve a specific organization
    - `POST /v2/organizations` - Create a new organization
    - `PUT /v2/organizations/:id` - Update an existing organization

    ### Rate Limits
    Zendesk Sell API has rate limits that vary by endpoint and subscription plan. Generally:
    - 600 requests per minute for most endpoints
    - Some endpoints have lower limits

    ### Authentication
    Zendesk Sell uses OAuth 2.0 for authentication. Nango handles the OAuth flow automatically.

    For more details, refer to the [official Zendesk Sell API documentation](https://developers.getbase.com/docs/rest/overview).
  </Tab>

  <Tab title="ðŸ”§ Connection Setup">
    <Steps>
      <Step title="Create a Zendesk Sell OAuth App">
        1. Log in to your Zendesk Sell account
        2. Go to **Settings** > **Customization** > **Integrations**
        3. Click on **Add Integration** and select **Custom Integration**
        4. Fill in the application details:
           - **Name**: Your application name
           - **Website URL**: Your application URL
           - **Redirect URI**: `https://api.nango.dev/oauth/callback`
        5. Save the integration and note the **Client ID** and **Client Secret**
      </Step>
      <Step title="Configure the integration in Nango">
        1. In the Nango dashboard, go to [Integrations](https://app.nango.dev/dev/integrations)
        2. Select **Zendesk Sell** and click **Configure**
        3. Enter your **Client ID** and **Client Secret** from step 1
        4. Click **Save**
      </Step>
      <Step title="Test the connection">
        1. Go to [Connections](https://app.nango.dev/dev/connections)
        2. Click **Add Test Connection**
        3. Select **Zendesk Sell** and click **Authorize**
        4. Log in to your Zendesk Sell account and grant permission
        5. You should see a successful connection
      </Step>
    </Steps>

## FAQ

<AccordionGroup>
  <Accordion title="What is Zendesk Sell?">
    Zendesk Sell is a CRM platform that helps sales teams track leads, manage deals, and close more sales. It provides tools for contact management, deal tracking, email integration, and sales analytics.
  </Accordion>

  <Accordion title="What data can I access through the Zendesk Sell API?">
    You can access deals, contacts, organizations, leads, activities, notes, tasks, and other sales-related data through the Zendesk Sell API.
  </Accordion>

  <Accordion title="Are there any rate limits?">
    Yes, Zendesk Sell has rate limits. The standard limit is 600 requests per minute, but this can vary based on your subscription plan and the specific endpoint.
  </Accordion>

  <Accordion title="Can I write data to Zendesk Sell through the API?">
    Yes, the Zendesk Sell API supports creating, updating, and deleting records including deals, contacts, and organizations.
  </Accordion>
</AccordionGroup>
