# Overview

To authenticate with Valley using an API Key, you need:
1. **API Key** - A key that grants Nango permission to interact with Valley's API resources and services.

This guide will walk you through obtaining your API key from Valley.

### Prerequisites:

- You must have a Valley account with Google sign-in.

### Instructions:

#### Step 1: Finding Your API Key

1. Sign in to your [Valley account](https://app.joinvalley.co/) and navigate to **Settings** in the Valley sidebar.
2. Navigate to the **Integrations** tab.
3. Click **Generate API Key** under **Valley API Key** to create your API key.

#### Step 2: Enter your credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Valley.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Valley (API Key).
