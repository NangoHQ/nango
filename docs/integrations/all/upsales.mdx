## API gotchas

- Upsales uses API token authentication via query parameter (`token`), not in headers.
- Rate limit is 200 requests per 10 seconds. The API returns `X-RateLimit-Reset` header when rate limited.
- Pagination is offset-based using `offset` and `limit` query parameters.
