## API gotchas

- Figma (SCIM) uses `API_KEY` auth mode with `Authorization: Bearer ${apiKey}` in the request header to access different endpoints.
- To establish a new connection in Nango, specify `www.figma.com` as the `host` `connectionConfig` for Figma organizations, and `figma-gov.com` for Figma for Government organizations.
