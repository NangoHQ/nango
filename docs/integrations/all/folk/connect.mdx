# Overview

To authenticate with Folk, you need:
1. **API Key** - A key that grants Nango permission to interact with Folk's API.

This guide will walk you through generating your API Key within Folk.

### Prerequisites:

- You must have a Folk account.

#### Step 1: Generate Your API Key
1. Navigate to [workspace settings](https://app.folk.app/apps/contacts/network/settings/api-keys).
2. Click **New API key**.
<img src="/integrations/all/folk/api_page.png" />
3. Enter a name for your API key, then click **Create key**.
<img src="/integrations/all/folk/create_key_form.png" />

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Folk.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/integrations/all/folk/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Folk.
