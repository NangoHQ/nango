# Overview

To authenticate with Cursor Admin, you need:
1. **API Key** - A key that grants Nango permission to interact with Cursor Admin API resources and services.

This guide will walk you through obtaining your API Key within Cursor.

### Prerequisites:

- You must have an account with Cursor with necessary permissions to generate an Admin **API Key**.

### Instructions:

#### Step 1: Finding Your API Key
1. Log in to your [Cursor](https://cursor.com/) account.
2. Navigate to [cursor.com/dashboard](https://cursor.com/dashboard) → **Settings** tab → **Advanced** → **Admin API Keys**.
3. Click **Create New API Key** to generate your **API Key**.

<Tip> Copy the generated key immediately, you won't see it again</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Cursor Admin.
2. Enter your **API Key** in its respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/integrations/all/cursor/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Cursor Admin.
