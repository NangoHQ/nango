# Overview

To authenticate with Cursor, you need:
1. **API Key** - A key that grants Nango permission to interact with Cursor's API resources and services.

This guide will walk you through obtaining your API Key within Cursor.

### Prerequisites:

- You must have an account with Cursor.

### Instructions:

#### Step 1: Finding Your API Key
1. Log in to your [Cursor](https://cursor.com/) account.
2. Navigate to the [Integrations](https://cursor.com/dashboard?tab=integrations) page.
3. In the **User API Keys** section, click the **New User API Key** button to generate your **API Key**.
<img src="/integrations/all/cursor/api_key.png"/>


#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Cursor.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.


<img src="/integrations/all/cursor/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Cursor.
