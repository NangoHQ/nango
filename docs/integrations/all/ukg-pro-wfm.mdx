## API gotchas
-  UKG Pro (Workforce Management) supports multiple [authentication](https://developer.ukg.com/wfm/docs/ukg-authentication-doc#ukg-authentication-application-flows-methods) methods, currently Nango supports the **Authorization Code flow**.
-  When a [tenant is provisioned](https://developer.ukg.com/wfm/docs/client-id-and-client-secret-doc), a password-protected Excel worksheet is attached to your company's Salesforce account that contains all of the details necessary to configure employees to make API calls. This information includes your **Tenant Name**, **Tenant URL**, **Client ID**, **Client Secret**, **SFTP** details, and so on.
