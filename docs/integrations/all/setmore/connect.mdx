# Overview

To authenticate with Setmore, you need:
1. **Access Token** - A unique initial token provided upon approval of your beta access request. It is used to generate an OAuth access token, which authenticates your API requests and refreshes it when it expires.

This guide will walk you through obtaining your **Access Token** within Setmore.

### Instructions:

#### Step 1: Requesting Access to the API
1. Sign up for a free Setmore account, if you haven’t already.
2. Send an email to **api@setmore.com** with the following details:
- Your full name
- The email address associated with your Setmore account
- A brief description of how you plan to use the API

<Note>Setmore reviews requests on a first-come, first-served basis. If selected, you’ll be contacted directly with instructions and provided an initial Access Token to begin the authentication process</Note>

#### Step 2: Enter credentials in the Connect UI

Once you have your **Access Token**:
1. Open the form where you need to authenticate with Setmore.
2. Enter your **Access Token** in its respective field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Setmore.
