## API gotchas

-  You can enable and disable available restful web service resources. Once a resource has been enabled, you can restrict its formats and authentication by clicking on its "Edit" link.
-  Nango currently supports Drupal rest authentication using the [Drupal REST & JSON API Authentication module](https://www.drupal.org/project/rest_api_authentication).
