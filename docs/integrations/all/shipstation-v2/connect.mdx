# Overview

To authenticate with ShipStation (v2), you need:
1. **API Key** - A key that grants Nango permission to interact with ShipStation (v2) API resources and services.

This guide will walk you through generating your API key within ShipStation.

### Prerequisites:

- You must set up an account with ShipStation.

#### Step 1: Generating an API Key
1. Sign in to your ShipStation account.
2. Navigate to **My Profile** > **Account** > **API Settings**, then select **V2 API** from the **Select API Version** dropdown.
<img src="/integrations/all/shipstation-v2/api_settings.png"/>
3. Click **Generate API Keys** to generate your **API Key**.
<Note>You can only generate one API Key at a time.</Note>


#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with ShipStation (v2).
2. Enter your **API Key** in the designated field.
3. Submit the form to complete authentication.


<img src="/integrations/all/shipstation-v2/form.png" style={{maxWidth: "450px" }}/>

You are now connected to ShipStation (v2).
