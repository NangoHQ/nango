## API gotchas

- Olostep uses API_KEY auth mode with `Authorization: Bearer <API-TOKEN>` in the request header.
- Common API endpoints include `POST /v1/scrapes`, `POST /v1/crawls`, and `POST /v1/answers`.
- A typical proxy configuration can target the Olostep API base URL and send the bearer token automatically.
```js
const config: ProxyConfiguration = {
    endpoint: '/v1/scrapes',
    headers: {
        Authorization: 'Bearer <API-TOKEN>',
        Accept: 'application/json'
    }
};
```
