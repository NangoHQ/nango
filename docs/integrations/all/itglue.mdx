## API gotchas

- All requests must include `Content-Type: application/vnd.api+json` header.
- API conforms to JSON API Spec with structured response format.
- Rate limit of 3000 requests per 5-minute window.
- Different base URLs for different domains: business.itglue.com.
- Maximum payload size is 10 MB.
- Dates are returned in RFC3339 format (UTC).
