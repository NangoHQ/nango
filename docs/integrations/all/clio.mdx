## API gotchas

-   No scopes are required for Clio's OAuth flow. Leave the scopes field empty.
-   Clio has regional API endpoints. The default configuration uses `app.clio.com` (US). For other regions, use `app.clio.eu` (EU), `app.au.clio.com` (AU), or `app.ca.clio.com` (CA) by overriding the authorization URL, token URL, and proxy base URL in your integration settings.
-   Access tokens expire after 7 days. Refresh tokens do not expire.
