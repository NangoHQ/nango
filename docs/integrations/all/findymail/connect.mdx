# Overview

To authenticate with FindyMail, you will need:

1. **API Key** - A key that is used to identify your FindyMail account.

This guide will walk you through finding these Keys within FindyMail.

### Prerequisites:

- You must have a registered FindyMail account.

### Instructions:

#### Step 1: Finding Your API Key

1. **Log in** to your FindyMail dashboard.
2. Click **API** in the left sidebar.
3. Create a new API key.
4. You can also find your API key at [FindyMail API](https://app.findymail.com/user/api-tokens)

#### Step 2: Enter API Key in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with FindyMail.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to FindyMail.
