## Connection Configuration

To connect to Nerdio through Nango, you need:

| Parameter | Description |
| - | - |
| Nerdio Instance URL | The hostname of your Nerdio Manager instance (e.g., `web-admin-portal-xxxxx.azurewebsites.net`) |
| Azure Tenant ID | Your Azure Active Directory Tenant ID |
| API Scope | The Application ID URI scope (e.g., `api://your-app-id/.default`) |
| Client ID | The Application (client) ID from Azure App Registration |
| Client Secret | The client secret from Azure App Registration |

For detailed setup instructions, see the [connection guide](/integrations/all/nerdio/connect).
