# Overview

To authenticate with FullEnrich, you will need:
1. **API Key** - A key that grants secure access to the FullEnrich API, enabling authorized applications to interact with FullEnrich resources and services.

This guide will walk you through obtaining your **API Key** from the FullEnrich API Dashboard.

### Prerequisites

- You must have a FullEnrich account.

### Instructions

#### Step 1: Getting your API key

1. Log in to your FullEnrich account and navigate to the [API Dashboard](https://app.fullenrich.com/app/api) page.
2. Copy or **Regenerate** an API key from this page.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with FullEnrich.
2. Enter your **API Key** in its respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/fullenrich/form.png" style={{maxWidth: "450px" }}/>

You are now connected to FullEnrich.
