# Overview

To authenticate with MillionVerifier, you need an **API Key**.

This guide walks you through locating your API key in MillionVerifier and connecting your account.

### Prerequisites

- A MillionVerifier account

### Instructions

#### Step 1: Finding your API key

1. Log in to your [MillionVerifier account](https://app.millionverifier.com).
2. Go to the [API page](https://app.millionverifier.com/api).
3. Copy your API key, or click **Add new API Key** to generate a new one.

<img src="/api-integrations/millionverifier/dashboard.png"/>

#### Step 2: Enter your credentials in the Connect UI

1. Open the form where you need to authenticate with MillionVerifier.
2. Enter your **API Key**.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/millionverifier/form.png" style={{maxWidth: "450px" }}/>

You are now connected to MillionVerifier.
