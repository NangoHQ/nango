# Overview

To authenticate with Quipteams, you will need:

1. **API Key** – A key that grants Nango permission to interact with Quipteams' API resources and services.

This guide explains how to obtain your **API Key** from Quipteams.

### Prerequisites

- An active Quipteams account.

### Instructions

#### Step 1: Finding your API key

1. Log in to your [Quipteams](https://quipteams.com) platform.
2. Navigate to **Company Settings** or the **Integrations** section.
3. Locate the API key management area and generate a new key.
4. Copy the key immediately — it is shown **once** at creation and cannot be retrieved later.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Quipteams.
2. Enter your **API Key** in the API Key field.
3. Submit the form to complete authentication.

<img src="/api-integrations/quipteams/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Quipteams.
