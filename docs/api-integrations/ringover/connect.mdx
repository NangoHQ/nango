# Overview

To authenticate with Ringover, you will need:

1. **API Key** – A key that grants secure access to the Ringover API, enabling authorized applications to retrieve phone, contact, user, and group information.

This guide explains how to obtain your **API Key** from Ringover.

### Prerequisites

- An active Ringover account with admin access

### Instructions

#### Step 1: Generating your API key

1. Open the [Ringover Dashboard](https://app.ringover.com).
2. Go to the **Developer** section.
3. Click the **Create an API key** button.
4. Select a user in the drop-down menu, enter a label for the key, then click **Create**.
5. Click the **Edit** button next to the new API key and select the appropriate rights for your use case.
6. Copy your API key and store it securely.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Ringover.
2. Enter your **API Key** in the API Key field.
3. Submit the form to complete authentication.

<img src="/api-integrations/ringover/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Ringover.
