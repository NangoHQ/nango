# Overview

To authenticate with Podscribe, you will need:

1. **API Key** - A key that grants access to the Podscribe API, enabling authorized applications to interact with your campaign and performance data.

This guide will walk you through generating your **API Key** in Podscribe.

### Prerequisites:

- You must have an active Podscribe account

### Instructions:

#### Step 1: Generating your API key

1. Log in to your Podscribe account.
2. Click your **initials** in the bottom left corner of the dashboard.
3. Select **Account** from the dropdown menu.

<img src="/api-integrations/podscribe/account.png"/>

4. Navigate to the **API** tab

<img src="/api-integrations/podscribe/api_tab.png"/>

5. Click **Generate Key** to create your API key
6. Copy the generated key and store it somewhere safe — you won't be able to see it again

<img src="/api-integrations/podscribe/generated_api_key.png"/>

<Tip>Keep your API key secure. If it's ever compromised, return to the API tab to regenerate it.</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Podscribe.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/podscribe/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Podscribe.
