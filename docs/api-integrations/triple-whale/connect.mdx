# Overview

To authenticate with the Triple Whale API via Nango, you will need:
1. **API Key** – An API key that is used to authenticate requests to the Triple Whale API.

This guide walks you through creating your **API Key**.

### Prerequisites

- A Triple Whale account.

### Instructions

#### Step 1: Generating Your API Key

1. Log in to your Triple Whale account, go to the [API Keys](https://app.triplewhale.com/api-keys) page, and click **Generate API Key**.

<img src="/api-integrations/triple-whale/api_key_page.png"/>

2. **Add a description** and **select scopes** based on your integration needs.

<img src="/api-integrations/triple-whale/scopes_page.png"/>

3. Click **Generate API Key**. Your new key will appear in a popup, **copy and store it securely**. Once the popup is closed, the key cannot be viewed again.


#### Step 2: Enter credentials in the Connect UI

Once you have your **API key**:

1. Open the form where you need to authenticate with Triple Whale.
2. Enter your **API key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/triple-whale/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Triple Whale.
