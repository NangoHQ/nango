## 🚀 Quickstart

Connect to Vercel via the integration install flow with Nango. Users install your integration in their Vercel team, granting team-scoped access to deployments, projects, env vars, domains, and more.

<Steps>
    <Step title="Create the integration">
    In Nango ([free signup](https://app.nango.dev)), go to [Integrations](https://app.nango.dev/dev/integrations) -> _Configure New Integration_ -> _Vercel (Integration)_.
    </Step>
    <Step title="Authorize Vercel">
    Go to [Connections](https://app.nango.dev/dev/connections) -> _Add Test Connection_ -> _Authorize_. Later, you'll let your users do the same directly from your app.
    </Step>
    <Step title="Call the Vercel API">
    Replace the placeholders below with your [secret key](https://app.nango.dev/dev/environment-settings), [integration ID](https://app.nango.dev/dev/integrations), and [connection ID](https://app.nango.dev/dev/connections):

    ```bash
    curl -X GET "https://api.nango.dev/proxy/v10/projects" \
      -H "Authorization: Bearer <NANGO-SECRET-KEY>" \
      -H "Provider-Config-Key: <INTEGRATION-ID>" \
      -H "Connection-Id: <CONNECTION-ID>"
    ```

    ✅ You're connected! Check the [Logs](https://app.nango.dev/dev/logs) tab in Nango to inspect requests.
    </Step>

    <Step title="Implement Nango in your app">
        Follow our [Auth implementation guide](/guides/primitives/auth#guide) to integrate Nango in your app.
    </Step>
</Steps>

## 📚 Vercel Integration Setup

1. Create your integration in the [Vercel Integration Console](https://vercel.com/dashboard/integrations/console)
2. Set the **Redirect URL** to your Nango callback URL (e.g. `https://api.nango.dev/oauth/callback`)
3. Configure **API Scopes** (deployment, project, project-env-vars, domain, team, user, etc.)
4. Copy the **Client ID** and **Client Secret** into your Nango integration settings
5. When creating connections, pass your integration's URL slug as the `slug` connection config parameter

Official docs: [Vercel Integration Documentation](https://vercel.com/docs/integrations/create-integration)
