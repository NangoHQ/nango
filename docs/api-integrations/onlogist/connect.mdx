# Overview

To authenticate with OnLogist, you will need:
1. **API Key** - A key that grants secure access to the OnLogist API, enabling authorized applications to interact with OnLogist resources and services.

This guide will walk you through requesting API access and finding your **API Key** within OnLogist.

### Prerequisites:

- You must have a registered OnLogist account.

### Instructions:

#### Step 1: Request API Access

To get your **API Key**, you need to request API access from OnLogist:

1. Email **[service@onlogist.com](mailto:service@onlogist.com)** to request API access for your account.
2. Once OnLogist processes your request, you will receive API access which will also include your **API Key**.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with OnLogist.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/onlogist/form.png" style={{maxWidth: "450px" }}/>

You are now connected to OnLogist.
