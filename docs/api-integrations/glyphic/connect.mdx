# Overview

To authenticate with Glyphic, you will need:
1. **API Key** - A key that grants secure access to the Glyphic API, enabling authorized applications to interact with Glyphic resources and services.

This guide will walk you through obtaining your **API Key** from the Glyphic dashboard.

### Prerequisites

- You must have a Glyphic account.

### Instructions

#### Step 1: Getting your API key

1. Log in to your Glyphic account and navigate to the [API Settings](https://app.glyphic.ai/settings/api) page.
2. Create or copy an API key from this page.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Glyphic.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/glyphic/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Glyphic.
