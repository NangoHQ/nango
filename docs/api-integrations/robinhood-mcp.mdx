## 🚀 Quickstart

Connect to the Robinhood Trading MCP server with Nango and start calling tools in minutes. Robinhood MCP uses OAuth 2.0 with **dynamic client registration**, so no app registration is required.

<Steps>
    <Step title="Create the integration">
        In Nango ([free signup](https://app.nango.dev)), go to [Integrations](https://app.nango.dev/dev/integrations) → _Configure New Integration_ → _Robinhood (MCP)_.
    </Step>
    <Step title="Authorize Robinhood">
        Go to [Connections](https://app.nango.dev/dev/connections) → _Add Test Connection_ → _Authorize_, then log in to your Robinhood account. Later, you'll let your users do the same directly from your app.
    </Step>
    <Step title="Call the Robinhood Trading MCP API">
        Make your first MCP request (initialize handshake). Replace the placeholders below with your [secret key](https://app.nango.dev/dev/environment-settings), [integration ID](https://app.nango.dev/dev/integrations), and [connection ID](https://app.nango.dev/dev/connections):
        <Tabs>
            <Tab title="cURL">

                ```bash
                curl "https://api.nango.dev/proxy/mcp/trading" \
                  -X POST \
                  -H "Authorization: Bearer <NANGO-SECRET-KEY>" \
                  -H "Provider-Config-Key: <INTEGRATION-ID>" \
                  -H "Connection-Id: <CONNECTION-ID>" \
                  -H "Content-Type: application/json" \
                  -d '{
                    "jsonrpc": "2.0",
                    "id": 1,
                    "method": "initialize",
                    "params": {
                        "protocolVersion": "2024-11-05",
                        "capabilities": {},
                        "clientInfo": {
                            "name": "nango-client",
                            "version": "1.0.0"
                        }
                    }
                  }'
                ```

            </Tab>
            <Tab title="Node">

                Install Nango's backend SDK with `npm i @nangohq/node`. Then run:

                ```typescript
                import { Nango } from '@nangohq/node';

                const nango = new Nango({ secretKey: '<NANGO-SECRET-KEY>' });

                const res = await nango.post({
                    endpoint: '/mcp/trading',
                    providerConfigKey: '<INTEGRATION-ID>',
                    connectionId: '<CONNECTION-ID>',
                    body: {
                        jsonrpc: '2.0',
                        id: 1,
                        method: 'initialize',
                        params: {
                            protocolVersion: '2024-11-05',
                            capabilities: {},
                            clientInfo: {
                                name: 'nango-client',
                                version: '1.0.0'
                            }
                        }
                    }
                });

                console.log(res.data);
                ```
            </Tab>
        </Tabs>

        ✅ You're connected! Check the [Logs](https://app.nango.dev/dev/logs) tab in Nango to inspect requests.
    </Step>
    <Step title="Implement Nango in your app">
        Follow our [quickstart](/getting-started/quickstart) to integrate Nango in your app.
    </Step>
</Steps>

## 📚 Robinhood (MCP) guides

Nango uses **dynamic client registration** with Robinhood's OAuth server. You do not need to register an MCP application — just create the integration in Nango and authorize with your Robinhood account.

Official resources: [Robinhood Trading MCP](https://robinhood.com/us/en/support/articles/agentic-trading-overview/)
