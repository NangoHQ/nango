# Overview

To authenticate with Dialpad WFM, you need:
1. **Access Token** — A bearer token that grants access to the Dialpad WFM API.

This guide walks you through obtaining your access token.

### Prerequisites

- An active Dialpad WFM account
- Access to a Dialpad WFM Customer Success Manager

### Step 1: Obtain your access token

Dialpad WFM access tokens are provisioned by your Customer Success Manager — they are not self-service.

1. Reach out to your **Dialpad WFM Customer Success Manager** and request an API access token.
2. Store the token securely once received. Treat it like a password — never expose it in client-side code, logs, or URLs.

<Note>If you do not have a Customer Success Manager contact, reach out to [Dialpad WFM support](https://help.dialpad.com) to request API access.</Note>

### Step 2: Enter credentials in the Connect UI

Once you have your access token:

1. Open the form where you need to authenticate with Dialpad WFM.
2. Enter your **Access Token** in the designated field.
3. Submit the form.

<img src="/api-integrations/dialpad-wfm/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Dialpad WFM.
