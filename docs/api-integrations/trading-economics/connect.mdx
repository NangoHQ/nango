# Overview

To authenticate with Trading Economics, you need:
1. **API Key** - A key that grants Nango permission to interact with Trading Economics' API resources and services.

This guide will walk you through getting your **API Key** from Trading Economics.

### Prerequisites

- A [Trading Economics developer account](https://developer.tradingeconomics.com/) subscribed to an API plan.

### Instructions

#### Step 1: Getting your API key

1. Go to the [Trading Economics developer portal](https://developer.tradingeconomics.com/) and subscribe to a plan.
2. Once subscribed, copy your API key from your account page.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Trading Economics.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/trading-economics/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Trading Economics.
