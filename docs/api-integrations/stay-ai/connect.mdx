# Overview

To authenticate with Stay AI, you will need:
1. **API Key** - Your Stay AI API key for authentication.

This guide will walk you through generating your **API Key** within the Stay AI Merchant Portal.

### Prerequisites:

- You must have a Stay AI merchant account ([sign up at stay.io](https://stay.io))

### Instructions:

#### Step 1: Create a new API Key

1. Log in to your [Stay AI Merchant Portal](https://stay.io).
2. Navigate to **Settings** -> **API Keys**.
3. Click the **New API Key** button.

<img src="/api-integrations/stay-ai/create_api_key.png"/>

4. Fill in the required fields: **Name**, **Description**, **Email**, and pick your desired **Scopes**.

<img src="/api-integrations/stay-ai/permissions.png"/>

5. Click **Create** to generate the key.
6. Copy the API key value shown and store it in a safe place.

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Stay AI.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.


<img src="/api-integrations/stay-ai/form.png" style={{maxWidth: "450px" }}/>


You are now connected to Stay AI.
