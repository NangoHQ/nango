# Overview

To authenticate with A-Leads, you need:
1. **API key** — A key that grants Nango permission to interact with the A-Leads API.

This guide will walk you through obtaining your API key from A-Leads.

### Prerequisites

- An active A-Leads account with API access.

### Step 1: Obtain your API key

1. Log in to your [A-Leads](https://app.a-leads.co) account.
2. Go to **Settings**.
3. Navigate to the **Integrations** tab.
4. In the **API Key** section, click **Create API Key**.
5. Assign permissions your API key needs.
6. Save the generated API key.

### Step 2: Enter credentials in the Connect UI

Once you have your API key:
1. Open the form where you need to authenticate with A-Leads.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/a-leads/form.png" style={{maxWidth: "450px" }}/>

You are now connected to A-Leads.
