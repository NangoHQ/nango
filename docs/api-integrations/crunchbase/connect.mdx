# Overview

To authenticate with Crunchbase, you will need:
1. **API Key** - Your Crunchbase API key for authentication.

This guide will walk you through obtaining your **API Key** for the Crunchbase API.

### Prerequisites:

- You must have a Crunchbase account with an Enterprise or Applications license that includes API access. Users on the Basic plan have access to [Basic APIs](https://data.crunchbase.com/docs/crunchbase-basic-using-api) only.

### Instructions:

#### Step 1: Get your API key

1. After registering for API access, your API key will be emailed to you.
2. If you have lost your key, contact [api@crunchbase.com](mailto:api@crunchbase.com) to have it resent.

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Crunchbase.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/crunchbase/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Crunchbase.
