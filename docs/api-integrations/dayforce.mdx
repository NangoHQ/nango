## ðŸš€ Quickstart

Connect to Dayforce with Nango and see data flow in 2 minutes.

<Steps>
    <Step title="Create the integration">
    In Nango ([free signup](https://app.nango.dev)), go to [Integrations](https://app.nango.dev/dev/integrations) -> _Configure New Integration_ -> _Dayforce_.
    </Step>
    <Step title="Authorize Dayforce">
    Go to [Connections](https://app.nango.dev/dev/connections) -> _Add Test Connection_ -> _Authorize_, then enter your Company ID, Username, and Password (and optional Client host / Identity host for test). Later, you'll let your users do the same directly from your app.
    </Step>
    <Step title="Call the Dayforce API">
    Replace the placeholders below with your [secret key](https://app.nango.dev/dev/environment-settings), [integration ID](https://app.nango.dev/dev/integrations), [connection ID](https://app.nango.dev/dev/connections), and your **Company ID** from the [connection config](/reference/api/connection/get):
    <Tabs>
        <Tab title="cURL">

            ```bash
            curl "https://api.nango.dev/proxy/api/{connectionConfig.companyId}/v1/employees" \
              -H "Authorization: Bearer <NANGO-SECRET-KEY>" \
              -H "Provider-Config-Key: <INTEGRATION-ID>" \
              -H "Connection-Id: <CONNECTION-ID>"
            ```

        </Tab>

        <Tab title="Node">

        Install Nango's backend SDK with `npm i @nangohq/node`. Then run (get `companyId` from the connection config via `getConnection`):

        ```typescript
        import { Nango } from '@nangohq/node';

        const nango = new Nango({ secretKey: '<NANGO-SECRET-KEY>' });

        const connection = await nango.getConnection('<INTEGRATION-ID>', '<CONNECTION-ID>');
        const companyId = connection.connection_config?.companyId as string;

        const res = await nango.get({
            endpoint: `/api/${companyId}/v1/employees`,
            providerConfigKey: '<INTEGRATION-ID>',
            connectionId: '<CONNECTION-ID>'
        });

        console.log(res.data);
        ```
        </Tab>


    </Tabs>
    Or fetch credentials with the [Node SDK](/reference/sdks/node#get-a-connection-with-credentials) or [API](/reference/api/connection/get).

    âœ… You're connected! Check the [Logs](https://app.nango.dev/dev/logs) tab in Nango to inspect requests.
    </Step>

    <Step title="Implement Nango in your app">
        Follow our [Auth implementation guide](/implementation-guides/platform/auth/implement-api-auth) to integrate Nango in your app.

        To connect with your own credentials, follow the connect guide linked below.
    </Step>
</Steps>
