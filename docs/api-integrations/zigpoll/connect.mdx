# Overview

To authenticate with Zigpoll, you will need:
1. **API Key** - A Zigpoll private API key that grants Nango access to your account.

### Prerequisites

- You must have a Zigpoll account. [Sign up](https://www.zigpoll.com) if you don't have one.

### Instructions

#### Step 1: Get your API token

1. Log in to the [Zigpoll Dashboard](https://www.zigpoll.com/app).
2. Click **Integrations** in the navigation.
3. Scroll down to **Private Keys** and click **Add Key**.
4. Give the key a label, then copy the generated key and store it securely.

#### Step 2: Enter your API key in the Connect UI

1. Open the form where you need to authenticate with Zigpoll.
2. Paste your API key into the **API Key** field.
3. Submit the form. You are now connected to Zigpoll.
