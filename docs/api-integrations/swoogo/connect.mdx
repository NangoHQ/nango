# Overview

To authenticate with Swoogo, you need:
1. **API Key** - Your Swoogo API key.
2. **API Secret** - Your Swoogo API secret.

This guide walks you through generating your Swoogo API credentials and connecting them to Nango.

### Prerequisites

- A Swoogo account.

### Instructions

#### Step 1: Generate your API credentials

1. Sign in to [Swoogo](https://www.swoogo.com/site/login).
2. Go to **My Profile** > **API Credentials**.
3. If no credentials have been generated yet, click **Generate credentials**.
4. Copy your **API key** and **API secret** and store them securely.

<img src="/api-integrations/swoogo/credentials.png"/>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API key** and **API secret**:

1. Open the form where you need to authenticate with Swoogo.
2. Enter your **API Key** and **API Secret** in their respective fields.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/swoogo/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Swoogo.
