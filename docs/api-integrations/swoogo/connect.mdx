# Overview

To authenticate with Swoogo, you will need:
1. **Consumer Key** - Your Swoogo API consumer key.
2. **Consumer Secret** - Your Swoogo API consumer secret.

Nango exchanges these for a short-lived bearer token using the OAuth 2.0 Client Credentials grant and refreshes it automatically.

---

### Step 1: Find your consumer key and secret

1. Log in to [Swoogo](https://www.swoogo.com/).
2. Go to **My Profile** > **API Credentials**.
3. Copy your **Consumer Key** and **Consumer Secret**.

<Info>
API access depends on your Swoogo plan. If you do not see the API Credentials section, contact your Swoogo account administrator.
</Info>

### Step 2: Enter credentials in the Connect UI

Once you have your credentials:
1. Open the form where you need to authenticate with Swoogo.
2. Enter your **Consumer Key** and **Consumer Secret** in their respective fields.
3. Submit the form - you should now be successfully authenticated.

You are now connected to Swoogo.
