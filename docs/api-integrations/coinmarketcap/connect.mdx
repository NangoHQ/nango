# Overview

To authenticate with the [CoinMarketCap API](https://coinmarketcap.com/api/documentation/v1/) via Nango, you will need:

1. **API key** – Your CoinMarketCap API key from [pro.coinmarketcap.com](https://pro.coinmarketcap.com/signup).

Requests to the production API use the header **`X-CMC_PRO_API_KEY`**. Nango stores your key as the connection credential and sends it on proxied requests.

### Prerequisites

- A CoinMarketCap account and API key (see [CoinMarketCap API overview](https://coinmarketcap.com/api/documentation/v1/)).

### Instructions

#### Step 1: Get an API key

1. Sign up at [pro.coinmarketcap.com](https://pro.coinmarketcap.com/signup) to receive your API key, as described in the [CoinMarketCap API documentation](https://coinmarketcap.com/api/documentation/v1/).
2. Copy your API key from the CoinMarketCap developer dashboard and store it securely.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API key**:

1. Open the form where you need to authenticate with CoinMarketCap.
2. Enter your **API key** in its respective field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to CoinMarketCap.
