# Overview

To authenticate with Autosana, you need:

1. **API Key** - A key that grants secure access to the Autosana API, enabling authorized applications to interact with Autosana resources and services.

This guide will walk you through getting your organization's **API Key** from Autosana.

### Prerequisites

- An active Autosana account.

### Instructions

#### Step 1: Getting your API key

1. Log in to Autosana and follow the welcome quickstart, or go to [**Settings → Integrations**](https://autosana.ai/settings?tab=integrations).
2. Copy your organization's API key.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Autosana.
2. Enter your **API Key** in the field provided.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/autosana/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Autosana.
