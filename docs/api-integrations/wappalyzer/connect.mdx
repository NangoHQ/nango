# Overview

To connect to the Wappalyzer API, you'll need:
1. **API Key** - A key that grants Nango permission to interact with Wappalyzer's API resources and services.

This guide walks through obtaining your API key.

### Prerequisites:
- An active Wappalyzer account at [wappalyzer.com](https://www.wappalyzer.com).
- An eligible plan with API access. See [Wappalyzer plans and pricing](https://www.wappalyzer.com/pricing) for details.


## Instructions

### Step 1: Get your API key
1. Log in to your Wappalyzer account and go to the **API key** tab in your account settings.
2. Copy the generated key.

### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Wappalyzer.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/wappalyzer/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Wappalyzer.
