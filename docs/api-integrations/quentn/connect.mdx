# Overview

To authenticate with Quentn, you need:
1. **API Key** - A key that grants Nango permission to interact with Quentn's API resources and services.
2. **API Base URL** - The base URL for your Quentn account's API endpoint.

This guide will walk you through obtaining both the **API Key** and **API Base URL** within Quentn.

### Prerequisites

- An active Quentn account.

### Instructions

#### Step 1: Getting Your API Key

1. Log in to your [Quentn](https://my.quentn.com) account.
2. Navigate to **Settings** > **API Info**.
3. Copy your **API Key**.

#### Step 2: Finding Your Base URL

1. In the **API Info** section from [Step 1](#step-1-getting-your-api-key), you will also find your **API Base URL**.
2. The base URL follows the pattern `https://<your-system>.quentn.com/public/api` (e.g., `https://xk7mdf.eu-5.quentn.com/public/api`).

#### Step 3: Enter credentials in the Connect UI

Once you have your **API Key** and **API Base URL**:

1. Open the form where you need to authenticate with Quentn.
2. Enter your **API Key** and **API Base URL** in their respective fields.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Quentn.
