## 🚀 Quickstart

Connect to ImmyBot with Nango and see data flow in 2 minutes.

<Steps>
    <Step title="Create the integration">
    In Nango ([free signup](https://app.nango.dev)), go to **Integrations** -> _Configure New Integration_ -> _ImmyBot_.

    This integration uses Microsoft Entra OAuth2 Client Credentials.
    </Step>
    <Step title="Authorize ImmyBot">
    Go to **Connections** -> _Add Test Connection_ -> _Authorize_, then enter your Microsoft Entra tenant ID, ImmyBot instance subdomain, client ID, and client secret. Later, you'll let your users do the same directly from your app.
    </Step>
    <Step title="Call the ImmyBot API">
    Make your first request to the ImmyBot API. Replace the placeholders below with your Nango API key from **Environment settings** > **API Keys**, integration ID, and connection ID:

    <Tabs>
        <Tab title="cURL">

            ```bash
            curl -X GET "https://api.nango.dev/proxy/api/v1/computers" \
              -H "Authorization: Bearer <NANGO-API-KEY>" \
              -H "Provider-Config-Key: <INTEGRATION-ID>" \
              -H "Connection-Id: <CONNECTION-ID>" \
              -H "Accept: application/json"
            ```

        </Tab>

        <Tab title="Node">

        Install Nango's backend SDK with `npm i @nangohq/node`. Then run:

        ```typescript
        import { Nango } from '@nangohq/node';

        const nango = new Nango({ secretKey: '<NANGO-API-KEY>' });

        const res = await nango.get({
            endpoint: '/api/v1/computers',
            providerConfigKey: '<INTEGRATION-ID>',
            connectionId: '<CONNECTION-ID>'
        });

        console.log(res.data);
        ```
        </Tab>
    </Tabs>

    Or fetch credentials with the [Node SDK](/reference/sdks/node#get-a-connection-with-credentials) or [API](/reference/api/connection/get).

    ✅ You're connected! Check the **Logs** tab in Nango to inspect requests.
    </Step>

    <Step title="Implement Nango in your app">
        Follow our [Auth implementation guide](/guides/primitives/auth) to integrate Nango in your app.

        To obtain your own production credentials, follow the setup guide linked below.
    </Step>
</Steps>
