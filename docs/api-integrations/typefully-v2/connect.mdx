# Overview

To authenticate with the Typefully Public API v2 via Nango, you need:
1. **API Key** – A key that grants Nango permission to call the Typefully v2 API.

This guide walks you through obtaining your **API Key**.

### Prerequisites

- A Typefully account.

### Instructions

#### Step 1: Generating your API key

1. Log in to your [Typefully](https://typefully.com) account.
2. Go to **Settings**.
3. Open the **API** section, click **+ New API Key**, enter a name, and submit. Your key will be generated; copy it and store it securely.

<Note>Use a key generated for API v2. Keys created for the legacy v1 API will not work with v2.</Note>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Typefully (API v2).
2. Enter your **API Key** in the designated field.
3. Submit the form; you should be successfully authenticated.

<img src="/api-integrations/typefully-v2/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Typefully API v2.
