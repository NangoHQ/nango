# Overview

To connect to the StoreCensus API, you'll need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with StoreCensus resources and services.

This guide walks through obtaining your API key.

### Prerequisites:
- An active StoreCensus account at [storecensus.com](https://www.storecensus.com).
- A Professional or Enterprise plan (required for API access).

## Instructions

### Step 1: Get your API key
1. Log in to your StoreCensus account and navigate to the [API Integration](https://www.storecensus.com/user/api) page.
2. Click **Generate API Key** and copy the key to your clipboard.

### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with StoreCensus.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/store-census/form.png" style={{maxWidth: "450px" }}/>

You are now connected to StoreCensus.
