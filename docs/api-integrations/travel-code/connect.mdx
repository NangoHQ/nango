# Overview

Travel Code uses the OAuth 2.0 authorization-code flow. To connect your Travel Code account, you only need an active Travel Code account — no keys or tokens to copy.

### Prerequisites:
- An active Travel Code account. If you don't have one yet, [book a demo](https://travel-code.com/) to get set up.

## Instructions

### Step 1: Start the authorization flow

1. Open the form where you need to authenticate with Travel Code.
2. Click **Authorize**. You'll be redirected to Travel Code's login page.

### Step 2: Log in and approve access

1. Log in with your Travel Code credentials.
2. Review the requested permissions (e.g. flight search, airport and airline lookups) and approve access.
3. You'll be redirected back automatically, and the connection is established.

You are now connected to Travel Code.
