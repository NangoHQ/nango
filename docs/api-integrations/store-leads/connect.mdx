# Overview

To authenticate with Store Leads, you will need:
1. **API Key** - Your Store Leads API key for authentication.

This guide will walk you through generating your **API Key** within the Store Leads dashboard.

### Prerequisites:

- You must have a Store Leads account ([sign up at storeleads.app](https://storeleads.app)) on a plan that includes API access.

### Instructions:

#### Step 1: Generate your API key

1. Log in to your [Store Leads account](https://storeleads.app/dashboard/account).
2. Navigate to the **API** tab on the Account page.
3. Generate a new API key and copy it.

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Store Leads.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/store-leads/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Store Leads.
