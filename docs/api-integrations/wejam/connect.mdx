# Overview

To authenticate with WeJam AI, you need:

1. **API Key** - A personal or organization key used to authenticate your requests to the WeJam AI API.

This guide will walk you through obtaining your API key from WeJam AI.

### Instructions

#### Step 1: Getting Your API Key

1. Log in and navigate to your [WeJam AI API Keys dashboard](https://auth.wejam.ai/org/api_keys/).
2. Choose the appropriate key type for your use case:
    - **Personal API Key** — for individual access tied to your user account.
    - **Organization API Key** — for shared access across your organization.
3. Click **New API Key** and submit the form.

<img src="/api-integrations/wejam/generate_api_key.png"/>

4. Copy your API key and store it securely, you won't be able to view it again.

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with WeJam AI.
2. Enter your **API Key** in its respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/wejam/form.png" style={{maxWidth: "450px" }}/>

You are now connected to WeJam AI.
