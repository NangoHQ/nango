# Overview

To authenticate with Lightfield, you need:
1. **API Key** - A dedicated Lightfield API key with the narrowest scopes required for your integration.

This guide walks you through creating a dedicated Lightfield API key and connecting it to Nango.

### Prerequisites

- A Lightfield account with permission to create API keys.

### Instructions

#### Step 1: Create an API key

1. Follow Lightfield's [API keys guide](https://docs.lightfield.app/using-the-api/api-keys) to create a new API key.
2. Create a dedicated key for this Nango integration instead of reusing a broader key.
3. Choose the narrowest scopes required for your use case. Review Lightfield's [scopes guide](https://docs.lightfield.app/using-the-api/scopes) before saving the key.
4. Copy the API key when it is shown and store it securely. Lightfield only displays a new key once, and you'll paste it into Nango during setup.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Lightfield.
2. Enter your **API Key** in the API Key field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Lightfield.
