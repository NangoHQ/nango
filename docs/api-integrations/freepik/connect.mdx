# Overview

To authenticate with Freepik, you need:
1. **API Key** - Your unique API key from the Freepik developer dashboard

This guide will walk you through finding or creating that credential within Freepik.


### Prerequisites

- You must have a Freepik account. Sign up at [freepik.com/api](https://freepik.com/api) if you don't have one.


#### Step 1: Obtain Your Freepik API Key

1. Log in to your Freepik account and navigate to the [API dashboard](https://freepik.com/developers/dashboard).

2. In the API key section, click the **Get API key** button to generate a new API key if you don't have one already.

<img src="/api-integrations/freepik/generate_api_key.png"/>

#### Step 2: Connect Your Freepik Account

Once you have your **API Key**:
1. Open the form where you need to authenticate with Freepik.
2. Enter your **API Key** in the corresponding field.
3. Submit the form. You should now be successfully authenticated.

<img src="/api-integrations/freepik/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Freepik.
