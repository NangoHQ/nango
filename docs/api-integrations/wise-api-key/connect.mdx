# Overview

To authenticate with Wise using an API Key, you will need:
1. **API Token** - A personal API token generated from your Wise Business profile.

This guide will walk you through generating your **API Token** within Wise.

### Prerequisites:

- You must have a Wise Business account.

### Instructions:

#### Step 1: Generating Your API Token

1. Log in to your [Wise account](https://wise.com).
2. Select your **Business profile**.
3. Navigate to **Your Account** -> **Integrations and Tools** -> **API tokens**.
4. Click **Add new Token**.
5. Complete 2-step verification.
6. Copy the generated token.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Token**:

1. Open the form where you need to authenticate with Wise.
2. Enter your **API Token** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/wise-api-key/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Wise.
