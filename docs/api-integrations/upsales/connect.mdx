# Overview

To authenticate with Upsales, you will need:

1. **API Token** - A token that is used to authenticate requests to the Upsales API.

This guide will walk you through finding your API Token within Upsales.

### Prerequisites:

- You must have a registered Upsales account with API access enabled.

### Instructions:

#### Step 1: Finding Your API Token

1. **Log in** to your Upsales account.
2. Click on your **profile icon** in the top-right corner.
3. Navigate to **Settings** or **Administration**.
4. Look for the **API** or **Integrations** section.
5. Your API Token should be displayed or you can generate one if not already created.

<Note>
If you don't see an API section, you may need to contact Upsales support or your account administrator to enable API access for your account.
</Note>

#### Step 2: Enter API Token in the Connect UI

Once you have your **API Token**:

1. Open the form where you need to authenticate with Upsales.
2. Enter your **API Token** in the designated field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Upsales.
