# Overview

To authenticate with Firstbase, you will need:
1. **API Key** - A key that grants secure access to the API, enabling authorized applications to interact with Firstbase resources and services.

This guide will walk you through generating your **API Key** within Firstbase.

### Prerequisites:

- You must have a Firstbase account with administrator access.

### Instructions:

#### Step 1: Generate your Firstbase API Key

1. Log in to the [Firstbase application](https://app.firstbasehq.com) as an administrator.
2. Navigate to the **Settings** menu in the top navigation bar.
3. Click on the **Firstbase API** tab.
4. Enter a name for your API key (e.g., "Development" or "Production") and click the button to generate it.

<Tip>Once your API key is generated, ensure you copy it to a secure location as it won't be shown again.</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Firstbase.
2. Enter your **API Key** in its designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/firstbase/form.png" style={{maxWidth: "450px" }}/>


You are now connected to Firstbase.
