# Overview

To authenticate with the Ordinal API, you will need:
1. **API Key** â€“ A key that grants secure access to the Ordinal API, allowing authorized applications to interact with Ordinal.

This guide walks you through generating and copying your **API Key** in Ordinal.

### Prerequisites

- An Ordinal account on a Pro plan or higher
- Access to workspace settings

### Instructions

#### Step 1: Generating your API key

1. Log in to your Ordinal workspace and navigate to the [Ordinal API](https://app.tryordinal.com/nango/settings/integrations/api) page.

<img src="/api-integrations/ordinal/api_key_page.png"/>

2. Click **+ Create an API Key** and provide a descriptive name for the key.
3. Copy the generated **API Key** immediately, as it is shown only once.

<img src="/api-integrations/ordinal/generated_api_key.png"/>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Ordinal.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/ordinal/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Ordinal.
