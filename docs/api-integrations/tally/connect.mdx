# Overview

To authenticate with Tally, you will need:

1. **API Key** â€” A key that grants secure access to the API and lets authorized applications interact with Tally resources and services.

This guide walks you through creating your **API Key** in Tally.

### Prerequisites

- An active Tally account

### Instructions

#### Step 1: Create an API key

1. Log in to your [API keys](https://tally.so/settings/api-keys) page.
<img src="/api-integrations/tally/overview.avif"/>

2. Click **Create API key**.
<img src="/api-integrations/tally/creating.avif"/>

3. Copy and store your **API Key** in a safe place. You will not be able to see it again after creation. If you lose it, create a new one.
<img src="/api-integrations/tally/created.avif"/>

<Tip>Currently, API keys are created in the context of your own user. Fine-grained permissions may be added by Tally in the future.</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Tally.
2. Enter your **API Key** in the respective field.
3. Submit the form; you should then be successfully authenticated.

<img src="/api-integrations/tally/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Tally.
