# Overview

Mercury supports both a **production** and a **sandbox** environment. You can connect to either when setting up your integration.

### Instructions

#### Step 1: Choose your environment

**Sandbox**

Enter the following hostnames to use the Mercury sandbox:

| Field | Value |
|---|---|
| OAuth Hostname | `oauth2-sandbox.mercury.com` |
| API Hostname | `api-sandbox.mercury.com` |

**Production (default)**

Leave the **API Hostname** and **OAuth Hostname** fields empty. Nango will connect to the production environment automatically.

#### Step 2: Authorize Mercury

1. Open the form where you need to authenticate with Mercury.
2. Optionally fill in the **API Hostname** and **OAuth Hostname** fields. Leave them empty for production.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/mercury/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Mercury.
