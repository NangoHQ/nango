This guide walks you through obtaining your API key from Gamma.

### Prerequisites

- You must have a Gamma Pro, Ultra, Team, or Business account.

### Instructions

#### Step 1: Getting your Gamma API key

1. Log in to Gamma and navigate to the [API Keys](https://gamma.app/settings/api-keys) page.
2. Click **Create API key** to generate a key.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API key**:
1. Open the form where you need to authenticate with Gamma.
2. Enter your **API key** in the field provided.
3. Submit the form; you should then be successfully authenticated.
