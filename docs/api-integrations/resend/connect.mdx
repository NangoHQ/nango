# Overview

To authenticate with Resend, you will need:

1. **API Key** – A key that grants secure access to the Resend API, enabling authorized applications to send email and manage Resend resources.

This guide explains how to obtain your **API Key** from Resend.

### Prerequisites

- An active Resend account

### Instructions

#### Step 1: Getting your API key

1. Log in to your Resend account and go to the [API Keys](https://resend.com/api-keys) page.
2. Click **+ Create API Key**.

<img src="/api-integrations/resend/api_keys_page.png"/>

3. Enter a name, select the permissions and domain the key has access to, then click **Add** to submit the form.

<img src="/api-integrations/resend/create_api_key_form.png"/>

4. Your API key will be generated. Store it securely—it is only displayed once.

<img src="/api-integrations/resend/created_api_key.png"/>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Resend.
2. Enter your **API Key** in the API Key field.
3. Submit the form to complete authentication.

<img src="/api-integrations/resend/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Resend.
