# Overview

To authenticate with WebinarJam, you need:

1. **API Key** - A key that grants Nango permission to interact with WebinarJam's API resources and services.

This guide will walk you through obtaining your **API Key** from WebinarJam.

### Prerequisites

- An active WebinarJam account on a paid plan (API access is not available on trial accounts).

### Instructions

#### Step 1: Getting Your API Key

1. Log in to your [WebinarJam](https://app.webinarjam.com/login) account.
2. Navigate to **Profile > API** and submit the API access application form.
3. Wait for approval (up to 2 business days).
4. Once approved, go to your **Webinars Dashboard**, click **Advanced** above any event, then click **API custom integration**.
5. Copy your **API Key**.

<Tip>Only one set of API credentials is generated per WebinarJam account. The same key works for all webinars.</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with WebinarJam.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

You are now connected to WebinarJam.
