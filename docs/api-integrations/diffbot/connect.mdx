# Overview

To authenticate with Diffbot, you need:
1. **API Token** — A token that grants Nango permission to interact with the Diffbot API.

This guide walks you through obtaining your API token from Diffbot.

### Prerequisites

- An active Diffbot account. You can [register for a free token](https://app.diffbot.com/get-started/) if you don't have one.

### Step 1: Obtain your API token

1. Log in to your [Diffbot dashboard](https://app.diffbot.com).
2. Click the link in the top right corner of the home screen.

<img src="/api-integrations/diffbot/api_key.png"/>

3. Your full API token is displayed on your token details page.

### Step 2: Enter credentials in the Connect UI

Once you have your API token:
1. Open the form where you need to authenticate with Diffbot.
2. Enter your **API Token** in the designated field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/diffbot/form.png" style={{maxWidth: "450px" }}/>


You are now connected to Diffbot.
