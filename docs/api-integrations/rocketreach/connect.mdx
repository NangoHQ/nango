# Overview

To authenticate with RocketReach, you will need:
1. **API Key** - Your RocketReach API key for authentication.

This guide will walk you through finding your **API Key** within the RocketReach account page.

### Prerequisites:

- You must have a RocketReach account.

### Instructions:

#### Step 1: Get your API key

1. Log in to your RocketReach account and navigate to the [API usage](https://rocketreach.co/account?section=nav_gen_api) page.
2. Copy your existing API key, or click **Generate New API Key** to create one.

<img src="/api-integrations/rocketreach/generate_api_key.png" />

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with RocketReach.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/rocketreach/form.png" style={{maxWidth: "450px" }}/>

You are now connected to RocketReach.
