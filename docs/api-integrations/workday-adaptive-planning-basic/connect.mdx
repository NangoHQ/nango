# Overview

To authenticate with Workday Adaptive Planning (Basic Auth), you will need:
1. **Username** - The username of the integration user
2. **Password** - The password for the integration user

### Prerequisites:

- An integration user set up in Adaptive Planning who has been assigned a role with appropriate permissions.

### Instructions:

#### Step 1: Find your Username and Password

- **Username**: The username of the integration user set up in Adaptive Planning.
- **Password**: The password for the integration user.

#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Workday Adaptive Planning (Basic Auth).
2. Enter your **Username** and **Password** in their respective fields.
3. Submit the form, and you should be successfully authenticated.

You are now connected to Workday Adaptive Planning (Basic Auth).
