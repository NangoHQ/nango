# Overview

To authenticate with Zero, you need:

1. **API Key** - A key that grants Nango permission to interact with the Zero API on your behalf.

This guide walks you through generating an API key in Zero.

### Prerequisites

- An active Zero account with access to workspace settings.

### Instructions

#### Step 1: Generate an API key

1. Log in to Zero and go to [Workspace Settings → API keys](https://app.zero.inc/settings/workspace/api).
2. Generate an API key and copy it.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with Zero.
2. Enter your **API Key** in the field provided.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/zero/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Zero.
