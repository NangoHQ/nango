# Overview

To authenticate with KlickTipp, you need:
1. **API Key** - A Listbuilding API key that grants Nango permission to interact with KlickTipp's subscriber API.

This guide will walk you through obtaining your **API Key** within KlickTipp.

### Prerequisites

- An active KlickTipp account.

### Instructions

#### Step 1: Getting Your API Key

1. Log in to your [KlickTipp](https://app.klicktipp.com) account.
2. Navigate to **Listbuilding** in the left sidebar.
3. Click **Create new listbuilding process** or select an existing one.
4. Choose **Entry via API key** as the listbuilding type.
5. Copy the **API Key** shown on the configuration page.

<Tip>Each API key is tied to a specific listbuilding process (subscription list). If you have multiple lists, each will have its own API key.</Tip>

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:

1. Open the form where you need to authenticate with KlickTipp.
2. Enter your **API Key** in the field provided.
3. Submit the form, and you should be successfully authenticated.

You are now connected to KlickTipp.
