# Overview

To authenticate with Granola, you will need:
1. **API Key** – Your Granola API key for authentication.

This guide will walk you through generating your **API key** in Granola.

### Prerequisites

- You must be a workspace administrator on the **Enterprise plan** (API access is available only on Enterprise).

### Instructions

#### Step 1: Generating your Granola API key

1. Sign in to your [Granola](https://granola.ai/) account.
2. Go to **Settings** → **Workspaces**.
3. Select the **API** tab.
4. Click **Generate API Key**.

#### Step 2: Enter credentials in the Connect UI

Once you have your **API Key**:
1. Open the form where you need to authenticate with Granola.
2. Enter your **API Key** in the respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/granola/form.png" style={{maxWidth: "450px" }}/>


You are now connected to Granola.
