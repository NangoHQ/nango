# Overview

To authenticate with Leadfeeder, you need:
1. **API Key** — A key that grants Nango permission to interact with the Leadfeeder API.

This guide walks you through obtaining your API key from Leadfeeder.

### Prerequisites

- An active Leadfeeder account with API access enabled. If you do not see the API keys section, contact your account administrator to grant access.

### Step 1: Obtain your API key

1. Log in to your [Leadfeeder](https://app.leadfeeder.com) account.
2. Navigate to **Settings** → **Personal** → **API Keys**.
3. Click **Create a new API key**, fill in the form, and copy the generated key.

<img src="/api-integrations/leadfeeder/create_api_key.png"/>

<img src="/api-integrations/leadfeeder/generated_key.png"/>

### Step 2: Enter credentials in the Connect UI

Once you have your API key:
1. Open the form where you need to authenticate with Leadfeeder.
2. Enter your **API Key** in the designated field.
3. Submit the form, and you should be successfully authenticated.


<img src="/api-integrations/leadfeeder/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Leadfeeder.
