# Overview

To authenticate with Canvas LMS, you will need:
1. **Canvas Install URL** - The base URL of your Canvas instance.

This guide will walk you through finding your **Canvas Install URL**.

### Prerequisites:

- You must have access to a Canvas LMS instance

### Instructions:

#### Step 1: Finding your Canvas Install URL

The Canvas Install URL is the base domain where your Canvas instance is hosted. This can be:

- **Instructure-hosted Canvas**: `canvas.instructure.com` or `your-school.instructure.com`
- **Self-hosted Canvas**: Your institution's custom domain (e.g., `canvas.yourschool.edu`)

To find your Canvas Install URL:

1. Log in to your Canvas LMS instance.
2. Look at the URL in your browser's address bar.
3. The domain (without the path) is your Canvas Install URL.

#### Step 2: Enter credentials in the Connect UI

Once you have your **Canvas Install URL**:

1. Open the form where you need to authenticate with Canvas LMS.
2. Enter your **Canvas Install URL** in its respective field.
3. Submit the form, and you should be successfully authenticated.

<img src="/api-integrations/canvas-lms/form.png" style={{maxWidth: "450px" }}/>

You are now connected to Canvas LMS.
