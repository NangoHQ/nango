# Overview

To authenticate with the Timetastic API, you will need an **API key**. Only admin users have access to an API key.

This guide walks you through generating one.

### Prerequisites

- A Timetastic account with admin access

### Instructions

#### Step 1: Generate your API key

1. Sign in to [Timetastic](https://app.timetastic.co.uk) as an admin.
2. Navigate to the [Integrations tab](https://app.timetastic.co.uk/api).
3. Select **Generate new key**. This replaces any existing key.
4. Copy the key.

<img src="/api-integrations/timetastic/dashboard.png"/>


#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Timetastic.
2. Enter your **API key** in its field.
3. Submit the form to connect your account.

<img src="/api-integrations/timetastic/form.png" style={{maxWidth: "450px" }}/>

You are now connected to the Timetastic API.
