# Overview

To authenticate with the Mandrill (Mailchimp Transactional) API, you will need an **API key**.

This guide walks you through generating one.

### Prerequisites

- A Mailchimp Transactional (Mandrill) account

### Instructions

#### Step 1: Generate your API key

1. Navigate to [mandrillapp.com/settings/index](https://mandrillapp.com/settings/index).
2. Select **+ Create API Key**.
<img src="/api-integrations/mandrill/dashboard.png"/>

3. Copy the key generated.


#### Step 2: Enter credentials in the Connect UI

1. Open the form where you need to authenticate with Mandrill.
2. Enter your **API key** in its field.
3. Submit the form to connect your account.

<img src="/api-integrations/mandrill/form.png" style={{maxWidth: "450px" }}/>

You are now connected to the Mandrill API.
